# kasflow
AI-powered personal expense tracker Telegram bot.

Tech stack:
- Python Telegram Bot
- LangGraph (for LLM-powered agents)
- OpenAI API (for LLM)

## Setup

Install dependencies:
```bash
make deps
```

Create a `.env` file:
```bash
cp .env.example .env
```


## Run

Run the bot:
```bash
make run
```
